```
Телеграм бот который фоткает расписание с сайта и отправляет его в чат 
не  забудьте добавить токен бота в файл .env 
в принципе не забудьте добавить все переменные окружения в файл .env
либо просто уберите его из кода и напишите везде свои данные в коде, 
если он не будет выкладыватьсяв гит

для того чтобы скачать telebof вам необходимо в папке .m2 создать файл settings.xml
и ввест там следующий код: 
```
```xml
<settings>
    <servers>
        <server>
            <id>github</id>
            <username>your_github_username</username>
            <password>your_github_password</password>
        </server>
    </servers>
</settings>
```
```
все остальные зависимости скачаются как обычно, но для селенмума вам нужно будет скачать драйвер
лиюо поднять в контейнере, но это уже другая история.
```
