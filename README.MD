```
Телеграм-бот, который фотографирует расписание с сайта и отправляет его в чат.
Не забудьте добавить токен бота в файл .env.
В принципе, не забудьте добавить все переменные окружения в файл .env,
либо просто уберите его из кода и напишите везде свои данные в коде,
если он не будет выкладываться в Git.
```
```xml
<settings>
    <servers>
        <server>
            <id>github</id>
            <username>your_github_username</username>
            <password>your_github_password</password>
        </server>
    </servers>
</settings>
```
```
все остальные зависимости скачаются как обычно, но для селенимума вам нужно будет скачать драйвер
либо поднять в контейнере, но это уже другая история.
```
